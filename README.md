## 📖 Sobre o projeto

O **Infra Lab** é um ambiente de aprendizado contínuo criado para centralizar estudos, laboratórios práticos, documentações técnicas e automações relacionadas ao universo da Infraestrutura de TI.

O principal objetivo deste repositório é reunir conhecimento adquirido durante estudos, cursos, projetos pessoais e experiências práticas, documentando todo o processo de forma organizada.

Além de servir como uma base de consulta pessoal, este projeto também busca compartilhar conhecimento com a comunidade e demonstrar a evolução técnica ao longo do tempo.

---

# 🎯 Objetivos

- Estudar Infraestrutura de TI
- Aprender DevOps na prática
- Criar laboratórios reproduzíveis
- Documentar procedimentos técnicos

- Explorar tecnologias modernas
- Compartilhar conhecimento

# 🎯 Filosofia do Projeto

Acredito que dominar os conceitos é mais importante do que conhecer apenas uma ferramenta específica.

Por isso, este repositório é organizado por áreas de conhecimento, utilizando diferentes tecnologias para colocar esses conceitos em prática.

Sempre que uma nova tecnologia é estudada, o objetivo é:

1. Entender os conceitos fundamentais.
2. Implementar um laboratório funcional.
3. Documentar todo o processo.
4. Registrar problemas encontrados e suas soluções.
5. Compartilhar o conhecimento adquirido.
